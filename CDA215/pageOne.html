Don't make me think

It means when a user looks at a page, it should be self-explanatory and glance for the visitor. User should be able to understand what it is and how to use it without spending time on thinking. Sometimes, if we are doing some new and very complicated page design, User may need to spend a little to understand, it is okay for that but it only takes a little time. If designer want the their own website to be effective, designer must show what their website's theme in the most obvious position that user can see them first when they entry a website.
